Configure Live Refresh
The Refresh button triggers a GitHub Actions run that pulls the latest data from Google Sheets directly — no waiting for the 2-hour cron. It needs a GitHub token to do this.
Create a fine-grained PAT at GitHub:
  1. Go to github.com/settings/tokens?type=beta
  2. Click Generate new token
  3. Repository access → Only select repositories → Daily-Team-use-Metrics
  4. Permissions → Repository → Actions → Read and write
  5. Generate & copy the token, paste below
